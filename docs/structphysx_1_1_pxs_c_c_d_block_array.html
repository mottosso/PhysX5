Detailed Description

template<typename T, int BLOCK_SIZE>
struct physx::PxsCCDBlockArray< T, BLOCK_SIZE >

a container class used in the CCD that minimizes frequency of hitting the allocator.

This class stores a set of blocks of memory. It is effectively an array that resizes more efficiently because it doesn't need to reallocate an entire buffer and copy data.
