Detailed Description

A structure to represent a potential CCD interaction between a pair of shapes.

◆ E_TOIType

enum physx::PxsCCDPair::E_TOIType

Defines whether this is an estimated TOI or an accurate TOI.

We store pairs in a priority queue based on the TOIs. We use cheap estimates to cull away work and lazily evaluate TOIs. This means that an element in the priority queue may either be an estimate or a precise result.
