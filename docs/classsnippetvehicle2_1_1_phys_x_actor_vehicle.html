Provide vehicle data items for this component.

Parameters
[out]axleDescriptionidentifies the wheels on each axle.
[out]commandsare the brake, throttle and steer values that will drive the vehicle.
[out]transmissionCommandsare the target gear and clutch values that will control the transmission. Can be set to NULL if the vehicle does not have a gearbox. If specified, then gearParams and gearState has to be specifed too.
[out]gearParamsThe gearbox parameters. Can be set to NULL if the vehicle does not have a gearbox and transmissionCommands is NULL.
[out]gearStateThe state of the gearbox. Can be set to NULL if the vehicle does not have a gearbox and transmissionCommands is NULL.
[out]engineParamsThe engine parameters. Can be set to NULL if the vehicle does not have an engine. Must be specified, if engineState is specified.
[out]physxActoris the PxRigidBody instance associated with the vehicle.
[out]physxSteerStateis the previous state of the steer and is used to determine if the steering wheel has changed by comparing with PxVehicleCommandState::steer.
[out]physxConstraintsThe state of the suspension limit and low speed tire constraints. If the vehicle actor is sleeping and constraints are active, they will be deactivated and marked as dirty.
[out]rigidBodyStateis the state of the rigid body used by the Vehicle SDK.
[out]wheelRigidBody1dStatesdescribes the angular speed of each wheel.
[out]engineStateThe engine state. Can be set to NULL if the vehicle does not have an engine. If specified, then engineParams has to be specifed too.
