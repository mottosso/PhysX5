Detailed Description

Class shared by all shape interactions for a pair of actors if contact reports are requested.

This class is used if at least one shape pair of an actor pair has contact reports requested.

Note
If a pair of actors had contact reports requested for some of the shape interactions but all of them switch to not wanting contact reports any longer, then the ActorPairReport instance is kept being used and won't get replaced by a simpler ActorPair instance.
