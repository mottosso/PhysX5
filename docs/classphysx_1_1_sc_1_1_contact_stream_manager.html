Detailed Description

Contact report logic and data management.

The internal contact report stream has the following format:

ContactStreamHeader | PxContactPairIndex0 | (PxContactPairPose0, PxContactPairVelocity0) | ... | PxContactPairIndexN | (PxContactPairPoseN, PxContactPairVelocityN) | (unused memory up to maxExtraDataSize ) | PxContactPair0 | ... | PxContactPairM | (unsued pairs up to maxPairCount)
