The entry point for the constraint solver.

Parameters
[in]dtThe simulation time-step
[in]continuationThe continuation task for the solver

This method is called after the island generation has completed. Its main responsibilities are: (1) Reserving the solver body pools (2) Initializing the static and kinematic solver bodies, which are shared resources between islands. (3) Construct the solver task chains for each island

Each island is solved as an independent solver task chain in parallel.
