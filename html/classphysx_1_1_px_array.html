Detailed Description

template<class T, class Alloc = typename PxAllocatorTraits<T>::Type>
class physx::PxArray< T, Alloc >

An array is a sequential container.

Implementation note entries between 0 and size are valid objects we use inheritance to build this because the array is included inline in a lot of objects and we want the allocator to take no space if it's not stateful, which aggregation doesn't allow. Also, we want the metadata at the front for the inline case where the allocator contains some inline storage space
