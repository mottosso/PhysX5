PxProfileNames  inNames,
uint32_t  inEventBufferByteSize

inlinevirtual

Create a new profile zone. This means you don't need access to a PxFoundation to create your profile zone object, and your object is automatically registered with the profile zone manager.

You still need to release your object when you are finished with it.

Parameters
inSDKNameName of the SDK object.
inNamesOption set of event id to name mappings.
inEventBufferByteSizerough maximum size of the event buffer. May exceed this size by sizeof one event. When full an immediate call to all listeners is made.
