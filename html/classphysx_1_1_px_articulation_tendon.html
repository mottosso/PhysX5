◆ release()

virtual void physx::PxArticulationTendon::release ( )
pure virtual

Releases a tendon to remove it from the articulation and free its associated memory.

When an articulation is released, its attached tendons are automatically released.

Note
Releasing a tendon is not allowed while the articulation is in a scene. In order to release the tendon, remove and then re-add the articulation to the scene.

Implements physx::PxBase.

Implemented in physx::NpArticulationSpatialTendon, and physx::NpArticulationFixedTendon.

virtual void physx::PxArticulationTendon::setOffset ( const PxReal  offset,
bool  autowake = true 
)
pure virtual

Sets the length offset term for the tendon.

An offset defines an amount to be added to the accumulated length computed for the tendon. It allows the application to actuate the tendon by shortening or lengthening it.

Parameters
[in]offsetThe offset term. Default: 0
[in]autowakeIf true and the articulation is in a scene, the call wakes up the articulation and increases the wake counter to PxSceneDesc::wakeCounterResetValue if the counter value is below the reset value.
See also
