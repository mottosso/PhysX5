eTEST_FOR_REMOVED_SHAPES 

Need to test stream for shapes that were removed from the actor/scene.

Usually this is the case when a shape gets removed from the scene, however, other operations that remove the broadphase volume of a pair object have to be considered as well since the shape might get removed later after such an operation. The scenarios to consider are:

  • shape gets removed (this includes raising PxActorFlag::eDISABLE_SIMULATION)
  • shape switches to eSCENE_QUERY_SHAPE only
  • shape switches to eTRIGGER_SHAPE
  • resetFiltering()
  • actor gets removed from an aggregate
eINVALID_STREAM 

Invalid stream memory not allocated.

eINCOMPLETE_STREAM 

Incomplete stream will be reported.

eNEEDS_POST_SOLVER_VELOCITY 

The stream contains extra data with PxContactPairVelocity items where the post solver velocity needs to get written to. Only valid for discrete collision (in CCD the post response velocity is available immediately).

Marker for the next available free flag.
